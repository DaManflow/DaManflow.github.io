Audit d'un site

Projet de BUT

Objectifs

Il s'agit d'un projet mené dans le cadre du BUT Informatique S5.L'objectif de ce projet était de développer en équipe une application de recherche de pays.
L'application a été développée avec le framework Express.js pour Node.js, ainsi que React Native.
Il a également été question de réaliser une vidéo de sensibilisation à la cybersécurité.

Compétences développées et mobilisées

  1. Réaliser : Améliorer une solution informatique pré-existante (mise en place d'une documentation, ajout d'éléments visuels notifiants, ajout d'un menu d'accessibilité)
  2. Optimiser : Optimiser les requêtes vers la base de données
  3. Gérer : Mettre en place une gestion de projet (suivi, planification, tableau des actions, etc.)
  4. Collaborer : Organiser et gérer le travail en équipe (outils de communication, plateforme de développement collaborative (Gitlab), répartition des rôles, etc.). Vulgariser le processus de l'audit afin de le rendre compréhensible par un plus large public (bande dessinée).
