Recherche de pays

Application de BUT

Objectifs

Il s'agit d'un projet mené dans le cadre du BUT Informatique S5.L'objectif de ce projet était de développer en équipe une application de recherche de pays.
L'application a été développée avec le framework Express.js pour Node.js, ainsi que React Native.
Il a également été question de réaliser une vidéo de sensibilisation à la cybersécurité.

Compétences développées et mobilisées

  • Réaliser : développer une application mobile avec un framework adapté, Programme National du BUT Informatique
  • Collaborer : organiser et gérer le travail en équipe (Gestion de projet, outils de communication, plateforme de développement collaborative (Gitlab), répartition des rôles, etc.)
  • Optimisation : les requêtes vers la base de données ont été pensées afin d'être les plus efficaces possibles, en réduisant leur complexité.
